Using the Simulation Template

The Simulation template is used to create interactive lesson pages and is designed make short work of software simulations.

SIMULATION TEMPLATE LAYERS

These are the layers in the simulation template. They should remain in this order.

screenshots: this layer will contain your imported screenshots.
incorrect: this layer should remain locked and not be changed.
hint: this layer contains an invisible movie clip named hint_mc which should be placed in the middle of your continue button.
instruction: this layer will contain your text boxes.
continue and continue2: these layers will contain your continue buttons. Continue buttons should be staggered on each layer.

Using the Simulation Template:

  1. Add graphics and text to the stage and control timing using the timeline.
  2. To have graphics or text fade in:
    1. Select the graphic or text and make it a Movie Clip using Modify > Convert to Symbol... (or ⌘-F8)
    2. In the Library panel, select the Movie Clip and open the Symbol Properties dialog by right-clicking and selecting Properties...
    3. Check Linkage: Export for ActionScript
    4. change Base Class to: com.davita.elements.FadeIn
  3. To pause the lesson and allow the student to read the screen, drag a ContinueBtn from the Library to the Stage.
  4. To end the lesson, drag a LessonEnded component from the Library to the Stage.
  5. Compile the swf using ⌘-return. (The Test fla does not need to be edited in Flash).
  6. Close your fla.
