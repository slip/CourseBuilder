Discover CourseBuilder

CourseBuilder was designed around a very specific workflow. We will walk you through that workflow step by step, demonstrating the creation of a new course. This is the way we use CourseBuilder ourselves, so we know that it works.

STEP 1 : CREATE A NEW COURSE

Create your course by clicking the New Course toolbar item.

Give the course a Course Code, Course Title, and Description.

Creating a New Course

STEP 2 : ADD COURSE SECTIONS

Create new sections for your course by clicking the New Section toolbar item.

Give each section a Section Title.

Adding a New Section

STEP 3 : ADD COURSE FILES

Create a new file by clicking the New File toolbar item or add an existing file by clicking the Add Existing toolbar item.

If creating a new file, choose a template and give each file a Filename, File Title, Keywords, and Description, then compile in Flash.

Create a New File

Add an Existing File

STEP 4 : PREVIEW YOUR COURSE

Preview your course by clicking the Preview toolbar item.

This will open your course in your default browser.

Preview Your Course

STEP 5 : BUILD YOUR COURSE

Build your course by clicking the Build toolbar item.

This will package your course for your LMS and save it to your desktop.

Build Your Course
