Using the Drag and Drop Template

The Drag and Drop template is used to create Drag and Drop quizzes.

Using the Drag and Drop Template:

  1. In your Library panel, edit dragBox 1-6.
  2. Also edit dropBox (note: dropBox will need a background, even if the background is transparent).
  3. On the Stage, make sure the instance names (in the properties panel) of your dragBoxes correspond to the correct dropBox.
  4. Compile the swf using ⌘-return and close your fla.
