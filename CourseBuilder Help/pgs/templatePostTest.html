Using the Post Test Template

The Post Test template is used to create LMS tracked True/False and Multiple Choice tests for your courses.

Using the Post Test Template:

  1. In CourseBuilder, edit Test Title and Passing Score.
  2. For each Question, edit the Question, Feedback and (optionally) Keywords. Add or remove questions using the + or - buttons beneath the Questions Table.
  3. For each Answer, edit the Answer and Correct checkbox. Add or remove answers (up to 4) using the + or - buttons beneath the Answers Table.
  4. Open the file in Flash using the Open in Flash button and compile the swf using ⌘-return. (The Test fla does not need to be edited in Flash).
  5. Close your fla.
